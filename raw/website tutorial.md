## How I Created and Deployed a Website From Scratch With Minimal/No Experience and Without Spending Any Money

Creating a website nowadays is as simple as explaining what you want in plain English — no complex code, no wrestling with HTML structure or CSS styling.
It's a matter of knowing exactly what you want and using AI such as Claude or ChatGPT.
When you want to create a personal website — or any website — you can simply have a conversation with an LLM and ask it to build one for you.
I created and deployed a website from scratch without writing a single line of code. That said, knowing how to code, or even just understanding basic file structure, will save you a lot of headaches down the line.
In this tutorial, I'll show you exactly how I created a website and deployed it online for free. Literally for free — no strings attached.
If you're wondering how to build a website because you need one, or you're simply curious, or you just want to know the simplest way possible — this is the one you're looking for.
So without further ado, let's get started.

Tools You'll Need
Here's everything you need for this project:

A Claude or ChatGPT account — you might already have one, but if not, you can easily sign up with your Google account
A web browser — any browser works
A code editor such as VS Code (optional) — useful if you want to directly edit your website's color scheme, personal information, or other content
A GitHub account — you can easily create one with your Google account

That's it. No fancy extras.

Step 1 — Generate Your Website
Open Claude or ChatGPT, then use this prompt:

Create a personal website using only HTML and CSS that is deployable to GitHub Pages. Here is my personal information:

Name: [your name]
Career: [your job or field]
Motto: [optional]
Interests: [your interests]
Skills: [your key skills]
Work history: [brief job history — you can paste your resume here, but remove any sensitive information]
Color scheme: [describe your preferred colors, or leave blank for a default]


Next, download the generated HTML file, then double-click it to open it in your browser. That's it — you now have a working personal website.

Step 2 — Make Improvements
If there's anything you want to change, just ask the AI. Here's an example prompt:

The header feels cramped. Can you increase the spacing between sections, make the header background dark navy (#0a192f) with white text, and bump the base font size to 16px?

This step is optional — if there's anything you'd prefer to edit directly, you can open the HTML file in VS Code or even a basic text editor like Notepad and make changes yourself.

Step 3 — Deploy It Online
Now for the final part. You want this website to be online so your friends can view it and compliment you on your genius. Assuming you're satisfied with how it looks, it's time to deploy.
There are many ways to host a website, but we're going with the easiest option — and it's completely free. (Hosting a website normally costs money, but not this way.)

Go to github.com and create an account — you can use Google sign-in
Choose your username carefully — it will appear in your website's URL
On the home page, click the + icon and select New repository
Set the visibility to Public — this is important
Name it personal-website or simply website
Upload your HTML file
Go to Settings → Pages, then under Source select Deploy from a branch → main → / (root)
Save — your website will be live within a few minutes

Your URL will look something like: https://yourusername.github.io/website
Ta-da! You now have a personal website that's viewable anywhere in the world.
